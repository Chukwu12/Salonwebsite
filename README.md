# Amour Hairstyle Studio — Client Website

A responsive, multi-page salon website built for **Amour Hairstyle Studio**, a full-service hair salon located in Jamaica, NY. The site serves as the salon's primary online presence, enabling clients to explore services, view a style gallery, and submit appointment requests.

**Live Site:** [amourhairstyles.netlify.app](https://amourhairstyles.netlify.app)  
**Built by:** OE Coders Corporation / Glofabs Enterprises

---

## Pages

| Page | Route | Description |
|---|---|---|
| Home | `/` | Hero section with background video, service highlights, and CTAs |
| Hairstyle / Services | `/products` | Full service menu, pricing, booking form, and style gallery |
| Contact | `/contact` | Contact info, appointment request form, and embedded Google Maps |
| Terms & Conditions | `/terms` | Legal terms for the salon |

---

## Features

- **Hero background video** on the homepage for an immersive first impression
- **Service menu** with pricing and estimated appointment durations across four categories:
  - Cuts & Barbering (from $45)
  - Scalp & Repair Care (from $55)
  - Color & Extensions (from $90)
  - Braids & Weaves (from $120)
- **Style gallery** organized by men's and women's hairstyles
- **Appointment booking form** on the services page (name, email, date, time, party size)
- **Contact page** with phone, email, address, hours, a contact form, and an embedded Google Maps location
- **Responsive design** — mobile-first layout across all pages
- **Accessible navigation** with a consistent header and footer across all pages

## Deployment

The site is deployed via **Netlify** with continuous deployment. Any pushes to the main branch will trigger an automatic redeploy.

To run locally, simply open `index.html` in your browser or use a local dev server:

```bash
# Using VS Code Live Server extension, or:
npx serve .
```
